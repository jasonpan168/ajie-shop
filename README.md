# Ajie Shop 商城系统

一个基于 PHP 的轻量级商城系统，适合个人或中小企业快速部署。

## ✅ 功能简介
- 商品分类、管理
- 订单创建、状态管理
- 后台配置、支付集成
- SQLite + 原生 PHP + Nginx 支持

## 🚀 快速开始

```
git clone https://github.com/jasonpan168/ajie-shop
cd ajie-shop
# 导入宝塔直接安装就行了
```
